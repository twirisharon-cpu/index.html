HTML as One of My Core Skills

HTML stands for HyperText Markup Language. It is not a programming language, but a markup language used to structure content on the web.

HTML uses elements (also called tags) to describe the meaning and role of content inside a web document.

1. Structural Elements

Tag What It Does When to Use It
<html> Wraps the entire HTML document Always used as the root element
<head> Contains metadata such as title and charset Used for non-visible information
<body> Contains all visible content Used for everything shown to users
<main> Main content area Used once per page

2. Semantic Elements

  • <header> – Intro content
  • <section> – Groups content
  • <article> – Independent content
  • <footer> – Closing section
  • <nav> – Navigation links

3. Text Formatting

  • <strong>
  • <em>
  • <u>
  • <br>
  • <hr>

4. Links & Images
